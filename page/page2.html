1. Développement Web

Python est largement utilisé pour créer des sites web et des applications web dynamiques. Le framework Flask permet de créer rapidement des applications simples.

Exemple de code (simplifier) :

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Bienvenue sur mon site web !"

if __name__ == '__main__':
    app.run(debug=True)

Ce code crée un serveur web de base qui affiche un message de bienvenue à l'utilisateur.

2. Analyse de Données

Python permet de manipuler et analyser des ensembles de données avec des bibliothèques comme pandas et numpy.

Exemple de code :

import pandas as pd

# Créer un DataFrame avec des données fictives
data = {'Nom': ['Alice', 'Bob', 'Charlie'], 'Âge': [23, 30, 35]}
df = pd.DataFrame(data)

# Afficher les données
print(df)

Ce code crée un tableau de données (DataFrame) et l'affiche à l'écran.

3. Automatisation de Tâches

Python est un excellent choix pour automatiser des tâches répétitives, comme la gestion de fichiers.

Exemple de code :

import shutil

# Copier un fichier d'un répertoire à un autre
shutil.copy('source.txt', 'destination.txt')

Ce code copie un fichier source.txt dans un autre répertoire sous le nom destination.txt.

4. Intelligence Artificielle et Apprentissage Automatique

Python est très utilisé dans le domaine de l'intelligence artificielle (IA) et du machine learning. Des bibliothèques comme scikit-learn permettent de créer des modèles de machine learning facilement.

Exemple de code :

from sklearn.linear_model import LinearRegression
import numpy as np

# Données d'exemple
X = np.array([[1], [2], [3], [4], [5]])  # Variable d'entrée
y = np.array([1, 2, 3, 4, 5])  # Variable de sortie

# Créer et entraîner le modèle
model = LinearRegression()
model.fit(X, y)

# Prédire une nouvelle valeur
prediction = model.predict([[6]])
print(prediction)  # Prédit la valeur de y pour X=6

Ce code crée un modèle de régression linéaire pour prédire une valeur de sortie basée sur une entrée donnée.

5. Développement de Jeux Vidéo

Python permet de créer des jeux vidéo 2D avec des bibliothèques comme pygame.

Exemple de code :

import pygame

# Initialiser pygame
pygame.init()

# Créer une fenêtre de jeu
screen = pygame.display.set_mode((400, 300))
pygame.display.set_caption("Mon jeu")

# Boucle de jeu
running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

# Quitter pygame
pygame.quit()

Ce code ouvre une fenêtre vide pour un jeu et attend que l'utilisateur la ferme.

6. Scripting et Gestion de Systèmes

Python peut être utilisé pour interagir avec le système d'exploitation, exécuter des commandes ou surveiller des processus.

Exemple de code :

import subprocess

# Exécuter une commande système
subprocess.run(['ls', '-l'])

Ce code exécute la commande ls -l pour afficher la liste des fichiers dans le répertoire courant.
