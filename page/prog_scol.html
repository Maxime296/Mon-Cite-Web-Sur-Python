Introduction

Ce cours est conçu pour vous aider à réviser Python, un langage essentiel du programme de seconde. Préparez-vous pour vos tests avec ces notions de base.

Syntaxe et Indentation

En Python, l'indentation définit les blocs de code. Contrairement à d'autres langages, il n'y a pas d'accolades. Une mauvaise indentation provoque une erreur d'exécution.

Les commentaires se font avec le symbole #. Tout ce qui suit est ignoré par le programme.

Exemple :

# Ceci est un commentaire
if x > 10:
    print("x est plus grand que 10")

Variables

Une variable est créée en lui affectant une valeur avec le symbole =. Python détermine automatiquement le type de la variable en fonction de la valeur assignée. Les types courants sont entier, flottant, chaîne de caractères et booléen.

Exemple :

x = 5  # Un entier
y = 3.14  # Un flottant
nom = "Alice"  # Une chaîne de caractères
est_majeur = True  # Un booléen

Conditions

Les structures conditionnelles permettent d'exécuter un bloc de code uniquement si une condition est vraie.

Exemple :

age = 20
if age >= 18:
    print("Majeur")
else:
    print("Mineur")

Boucles

Les boucles permettent de répéter un bloc de code plusieurs fois :

  • for : permet de répéter un bloc un nombre défini de fois.
  • while : répète le bloc tant qu'une condition est vraie.

Exemple avec for :

for i in range(5):
    print(i)

Exemple avec while :

compte = 0
while compte < 5:
    print(compte)
    compte += 1

Exercice : Moyenne des notes

Voici un script simple pour calculer la moyenne des notes :

# Demander le nombre de notes
nombre_notes = int(input("Nombre de notes : "))

# Initialiser la somme
somme = 0

# Boucle pour entrer les notes
for i in range(nombre_notes):
    note = float(input(f"Note {i + 1} : "))
    somme += note

# Calculer et afficher la moyenne
moyenne = somme / nombre_notes
print(f"Votre moyenne est : {moyenne}")

Questions Fréquentes

  • Type de x après x = 3.5 :

    Le type de x est flottant (ou float en anglais). En Python, lorsqu'un nombre décimal est assigné à une variable, celle-ci devient automatiquement un nombre à virgule flottante (type float).

    Exemple :

            x = 3.5  # x devient un flottant
        print(type(x))  # Affiche
  • Valeur de x après x = 10; x = x + 5 :

    La valeur de x sera 15. Voici pourquoi :

    Dans cette opération, x = 10 assigne la valeur 10 à x, puis l'expression suivante x = x + 5 ajoute 5 à la valeur actuelle de x (qui était 10), donc x devient 15.

    Exemple :

            x = 10
        x = x + 5  # x devient 15
        print(x)  # Affiche 15
  • Sortie de print("informatique"[1:4]) :

    La sortie de cette commande sera "nfo".

    En Python, lorsque vous utilisez une tranche (slice) sur une chaîne de caractères, vous obtenez une sous-chaîne allant de l'indice de début (inclus) jusqu'à l'indice de fin (exclus). Ici, [1:4] signifie que l'on commence à l'indice 1 et on s'arrête avant l'indice 4. La chaîne "informatique" contient les caractères suivants :

    • Indice 0 : "i"
    • Indice 1 : "n"
    • Indice 2 : "f"
    • Indice 3 : "o"
    • Indice 4 : "r"

    Les caractères compris entre l'indice 1 et 3 sont "n", "f" et "o", d'où la sortie "nfo".

    Exemple :

            print("informatique"[1:4])  # Affiche "nfo"
  • Nombre de fois que "Python" est imprimé dans for i in range(5): print("Python") :

    Le mot "Python" sera imprimé 5 fois.

    Le range(5) génère une séquence de 5 valeurs allant de 0 à 4, et à chaque itération, la fonction print("Python") est appelée, affichant le mot "Python".

    Exemple :

            for i in range(5):
            print("Python")  # Affiche "Python" 5 fois
