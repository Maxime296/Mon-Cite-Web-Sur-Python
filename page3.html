Prérequis

Avant de commencer, assurez-vous d'avoir installé Python et PIP sur votre ordinateur cliquez ici pour un tuto

Vous aurez également besoin d'un éditeur de texte pour écrire vos scripts. Vous pouvez utiliser n'importe quel éditeur de texte, mais nous vous recommandons d'utiliser un éditeur de code comme Visual Studio Code (flexible), Notepad++ (logiciel libre) ou PyCharm (spécialement pour Python).
